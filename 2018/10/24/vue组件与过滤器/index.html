vue组件与过滤器

  • 2018-10-24
  • Lmz

Vue 组件

父与子，使用者是父
父需要声明子组件，引入子组件对象，声明方式：
import 子组件对象 from ‘../xxx.vue’;
{
components:{
组件名：子组件对象
}
}

全局组件：
不需要声明，直接用。
vue.component(‘组件名’, 组件对象)

组件之间的通信：
1.父——>子
父：常量或变量 子：根属性props:['prop1','prop2'],
页面中直接用,js中用this.prop1获取
2.子——>父（略）
父监听，父子共用电话，当子有需求触发通信

API简析
全局：Vue.
实例：this/new Vue()
选项：new Vue()的参数/export default里面的属性

Vue 过滤器

vue中无内置过滤器，只有自定义过滤器。分为组件内的过滤器，全局过滤器。
1.组件内过滤器
选项中的一个filters的属性（一个对象）key:过滤器名；value:过滤方式函数体
2.全局过滤器
Vue.filter(过滤器名,fn)
